# 自然教室座位安排

直接開啟 `index.html` 即可使用，不需要安裝任何套件。

## 功能

- 手動新增、編輯、刪除學生資料
- 學生資料包含座號、姓名、性別
- 支援基礎 CSV 匯入
- 分組模式：最多 6 組，每組 6 座位，可設定組數、空桌、男女座位配置
- 分組模式目前採動態規則：每組先以 2 男 2 女為基準，再依 6→5→4→3→2→1 的順序增減
- 排列模式：固定 6 排 x 5 座位
- 自動排座與重新排列
- 座位不足、男女座位需求不符時會顯示提示

## CSV 格式

可使用以下格式：

```csv
座號,姓名,性別
1,王小明,男
2,林小美,女
```

也接受：

```csv
number,name,gender
1,Tom,male
2,Amy,female
```

## 使用方式

1. 開啟 `index.html`
2. 在左側輸入學生名單，或匯入 CSV
3. 在上方切換分組模式或排列模式
4. 依需求調整座位設定
5. 按下「自動排座」完成安排
6. 若想重新洗牌，按「重新排列」
